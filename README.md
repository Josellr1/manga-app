# MangaZen 📖

Tu lector de manga personal en español, construido con la API de MangaDex.

## 🚀 Ver la app

👉 **[Abrir MangaZen](https://TU_USUARIO.github.io/manga-app/)**

## ✨ Funcionalidades

- 🔥 Miles de mangas populares
- 🇪🇸 Capítulos en español (con respaldo en inglés)
- 🔍 Búsqueda en tiempo real
- 🎭 Filtros por género
- 📖 Lector vertical o paginado
- 💜 Biblioteca de favoritos
- ✅ Historial de capítulos leídos

## 🛠️ Tecnología

- HTML + CSS + JavaScript vanilla
- API: [MangaDex](https://api.mangadex.org)
- PWA instalable en móvil
